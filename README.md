# git-assignment-kevin-bedward

About This Assignment

This project showcases the foundational Git skills required for modern QA workflows.
Using Git is essential for:

Collaborating with developers
Tracking changes to test cases and documentation
Managing test automation code
Contributing to version-controlled QA repositories
Working in Agile and DevOps teams

This repository contains screenshots and steps from my Git learning assignment

##Topics Covered
- Cloning repositories
- Committing changes
- Pushing to remote
- Creating branches
- Merging
- Handling conflicts
- Viewing commit history
